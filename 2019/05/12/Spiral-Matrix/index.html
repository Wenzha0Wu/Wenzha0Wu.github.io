Spiral Matrix

Posted on 2019-05-12 | In fundamental

problem description

  • given two integers m and n , return a matrix with $n \times m$ elements (n rows, m columns), all elements form 1 to m*nof the matrix in spiral order.($1 \leq m,n \leq 100$)

  • example:

    input :

    3 3

    output :

    1 2 3

    8 9 4

    7 6 5

  • problem url

solution

  • using dx[] = {0, 1, 0, -1}, dy[] = {1, 0, -1, 0} to present four directions
  • when it comes to corner, let d = (d + 1) % 4
